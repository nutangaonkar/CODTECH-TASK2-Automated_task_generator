# CODTECH Internship Task 2
## Automated Report Generation

### Description
This project reads student data from a CSV file, analyzes it, and generates a formatted PDF report automatically using Python.

### Technologies Used
- Python
- CSV module
- FPDF library

### Files
- marks.csv → Input data file
- report_generator.py → Python script
- student_report.pdf → Generated report

### How to Run
1. Install library:
   pip install fpdf
2. Run the script:
   python report_generator.py
